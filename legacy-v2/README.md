# Kweli — Truth Infrastructure  
*A verification layer by EAX Global*

Kweli is a lightweight verification infrastructure designed to prove the authenticity and integrity of certificates, reports and environmental records.

The system allows organisations to:
- Hash documents client-side
- Anchor proofs on-chain
- Generate public verification links and QR codes
- Instantly verify authenticity without exposing private data

---

# What Problem Does Kweli Solve?

Across environmental markets, exports, labs, ESG reporting and institutional systems, trust is fragmented.

Documents can be:
- altered
- duplicated
- spoofed
- unverifiable
- difficult to audit

Kweli creates a permanent proof layer underneath existing systems.

Instead of replacing workflows, Kweli verifies them.

> JSON runs it. Blockchain proves it.

---

# Core Verification Flow

1. A certificate/report is created in an existing system
2. The document is hashed locally using SHA-256
3. The hash is anchored on-chain
4. A QR code / verification URL is generated
5. Anyone can scan and instantly verify authenticity

No raw document data is stored publicly.

Only proof of integrity is recorded.
